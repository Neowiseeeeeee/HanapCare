# Build HanapCare - Complete Hospital Management System

You are a senior full-stack software architect, UI/UX designer, database engineer, DevOps engineer, healthcare systems analyst, and product manager.

Your task is to design and generate a production-quality MVP called:

# HanapCare

Modern Hospital Management System for the Philippines

The system must be built as a complete full-stack web application with all pages, database models, UI components, workflows, navigation, dashboards, sample data, role permissions, forms, and business logic already connected.

Generate the application in one cohesive architecture rather than creating isolated modules.

---

## Project Vision

HanapCare is a cloud-based Hospital Management System designed for hospitals, clinics, medical centers, and healthcare facilities in the Philippines.

The goal is to digitize hospital operations from patient registration to billing while maintaining security, scalability, compliance, and usability.

The application should feel modern, professional, enterprise-grade, and comparable to commercial healthcare platforms.
